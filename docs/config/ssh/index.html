配置全局Git的SSH

@Jenkins版本:2.206

jenkins从代码版本控制服务器上拉取代码，使用ssh key的方式配置git，不需要使用账户/密码模式来配置git

关于git的安装配置见Git Doc - Jaswine

1.1.生成密钥

# 在安装Jenkins的电脑上运行
ssh-keygen

1.2.配置公钥

将./ssh/id_rsa.pub 中的内容复制好，配置到git服务器中的ssh key那。[下图是Github上的例子]

1.3.验证是否成功

运行git clone xxx.git是否能下载，可以把代码下载下来就说明是ok的
